# Motion2Motion Project Page

这是为SIGGRAPH Asia 2025论文"Motion2Motion: Cross-topology Motion Transfer with Sparse Correspondence"设计的项目页面。

## 功能特点

- 🎨 现代、响应式设计
- 📱 移动设备友好
- 🌙 深色模式支持
- 📋 一键复制BibTeX引用
- 🎥 视频展示区域
- 📊 结果展示网格
- 🔗 相关链接集成

## 文件结构

```
project page/
├── index.html          # 主页面文件
├── responsive.css       # 响应式样式增强
└── README.md           # 说明文档
```

## 使用说明

1. **部署页面**: 将 `index.html` 文件部署到任何静态网站托管服务（如 GitHub Pages、Netlify、Vercel 等）

2. **更新内容**: 
   - 替换视频占位符为实际的演示视频
   - 更新论文PDF链接
   - 添加实际的结果图片/视频
   - 更新代码仓库链接

3. **自定义样式**: 
   - 修改 CSS 变量来调整颜色方案
   - 添加自定义动画效果
   - 调整布局和间距

## 需要替换的占位符内容

- [ ] **视频**: 添加项目演示视频
- [ ] **论文PDF**: 更新论文下载链接
- [ ] **结果图片**: 替换结果展示区域的占位符
- [ ] **代码链接**: 确认GitHub仓库链接
- [ ] **Blender插件**: 添加插件下载链接
